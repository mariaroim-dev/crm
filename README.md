# CRM System

Final educational JavaScript project.
A CRM system built according to a Figma design with full responsive layouts.
The project focuses on complex JavaScript logic, dynamic data handling, and UI interaction.

## Project Overview
This application represents a simple CRM system for managing clients.
The interface is fully implemented according to the design layout and adapted for different screen sizes.
The project demonstrates working with structured data, user interactions, and application state management.

## Design
Figma design with adaptive layouts:
https://www.figma.com/design/oimAXuLHLFaKoTYCz8U20s/CRM--Copy-?node-id=211-746

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Flexbox
- Responsive layout

## Features
- Dynamic rendering of client data
- Add new clients
- Edit existing clients
- Delete clients
- Sorting and filtering
- Search functionality
- Form handling and validation
- Interactive UI without page reload
- Fully responsive layout based on the design

## How to Run the Project Locally

1. Clone the repository and navigate to the project folder:
```bash
git clone https://github.com/mariaroim-dev/crm.git
cd crm
```
2. Install dependencies:
```
npm install
```
3. Start the project:
```
npm run start
```
(Use the exact script name defined in package.json.)

If the project does not use a dev server, open the main HTML file in your browser
or run a local server such as:
```
npx serve
```
